## Features

- **Five explorable zones** with distinct atmospheres and environmental lore
- **Eight NPCs** — each with their own story, secrets, and night-time behaviors
- **Dynamic weather moods** that shift per zone over time
- **Journal system** with filters (`All`, `Visited`, `Secrets`) and dual-page layout
- **Forum Archive** — thread list + post viewer panel with in-world posts
- **Three distinct endings** based on player choices
- **Photo Mode** with in-game screenshot capture
- **Procedural art** — no external image assets, everything is drawn at runtime
- **Accessibility presets** — High Contrast, Calm Motion, Large Text, Colorblind Palette
- **Save slots** with export/import JSON — your progress, your file

## Getting Started

No build tools. No install step. Just open and play.

```bash
git clone https://github.com/jordannealexie/the-last-modder.git
cd the-last-modder
# Open index.html in Chrome, Edge, or Firefox
```

> **Tip:** Interact with the game once after page load to initialize Web Audio.

## Controls

| Action | Key |
|---|---|
| Move | `↑ ↓ ← →` or `W A S D` |
| Interact / Advance Dialogue | `E` or `Click` |
| Journal | `J` |
| Credits | `C` |
| Photo Mode | `P` |
| Screenshot (in Photo Mode) | `O` |
| Hotkeys Overlay | `H` |
| Changelog Panel | `L` |
| Performance Overlay | `I` |
| Close Panels | `Esc` |

---

## Settings

Open the `SET` panel in-game to access:

- Volume control
- Dialogue speed
- Large text mode
- CRT overlay toggle
- Colorblind-friendly palette
- Accessibility presets (`High Contrast`, `Calm Motion`)
- Save slot switching
- Save export / import (JSON)

---

## Tips

- Stand near environmental objects and press `E` to inspect hidden lore
- Some NPCs shift position at **night** — revisit zones after dark
- Weather moods change over time and affect zone atmosphere
- Journal **Secrets** filter reveals entries you might have missed
